This web app demonstrates how a single-layer neural network - also known as a perceptron - works. Think of the inputs as a line of light hitting the retina - a neuron only fires if there is light hitting it. The goal is for the network to distinguish between the two inputs, so that it will output 0 for the input pattern on the left, and output 1 for the input pattern on the right.
