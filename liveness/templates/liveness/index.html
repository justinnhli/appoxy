Liveness Analysis Demo (Allen and Cooke, 1976)

Supported syntax: non-negative integer values; lower-case variables and functions; numeric and boolean operators; braced if, if-else, and while.

Unsupported syntax: string and decimal values; negation (-); not (!); bit operators; else if, for, do-while; break, continue.
